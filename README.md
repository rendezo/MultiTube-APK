# 🇬🇧 English

## ✨ Key Features
- **Quad-Stream Layout**: Watch up to 4 live streams simultaneously.
- **TV Remote Optimized**: Seamless navigation designed specifically for standard Android TV remotes.
- **Smart Quality Control**: Auto-adjusts resolution based on focus and TV decoder capabilities.
- **Long-Press Refresh**: Stuck stream? Just long-press the center button to re-extract and reload.
- **One-Click Deployment**: Built-in automated update system to keep you on the latest version.

## 🎮 Remote Control Legend
- **DPAD Center (OK)**: 
  - *Short Press*: Toggle Fullscreen (Zoom in/out).
  - *Long Press*: **Refresh Slot** (Re-loads the current stream if stuck).
- **DPAD Up / Down**: Adjust volume for the focused window.
- **DPAD Left / Right**: Navigate focus between the 4 windows.
- **MENU / BACK**: Toggle the control panel & settings.

## 🚀 Installation
1. Download the latest APK from here: [**MultiTube.apk**](https://github.com/rendezo/MultiTube-APK/raw/main/MultiTube.apk)
2. Install it on your Android TV (using a Pendrive or "Send Files to TV" app).
3. Enter your video IDs or load your saved slots and enjoy!

# 🇭🇺 Magyar leírás

## ✨ Főbb jellemzők
- **4-Ablakos elrendezés**: Nézz akár 4 streamet egyszerre, egyetlen képernyőn.
- **TV Távirányítóra optimalizálva**: Nincs szükség egérre, minden funkció elérhető a tavirányítóval.
- **Intelligens minőségvezérlés**: Automatikusan állítja a felbontást a fókusz és a TV hardvere alapján.
- **Hosszú nyomásos frissítés**: Elakadt az adás? Csak tartsd hosszan az OK gombot az újratöltéshez.
- **Automatikus frissítés**: Mindig jelzi, ha elérhető új verzió a folyamatos fejlesztésekhez.

## 🎮 Távirányító használata
- **Középső gomb (OK)**: 
  - *Rövid nyomás*: Teljes képernyő (Be/Ki).
  - *Hosszú nyomás*: **Ablak frissítése** (Újraindítja az adott ablakot).
- **Fel / Le nyíl**: Hangerő állítása a kiválasztott ablakban.
- **Bal / Jobb nyíl**: Ablakok közötti váltás (fókusz).
- **MENU / VISSZA**: Vezérlőpanel megjelenítése és elrejtése.

## 🚀 Telepítés
1. Töltsd le a legfrissebb verziót innen: [**MultiTube.apk**](https://github.com/rendezo/MultiTube-APK/raw/main/MultiTube.apk)
2. Másold át a TV-re (pendrive-on vagy hálózaton keresztül).
3. Add meg a videó ID-kat, mentsd el a kedvenceidet és kész!
